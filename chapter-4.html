Introduction

In this chapter, we will discuss Django templates and the methods Django provides to deal with the templates. Templates are simply plain text files that can be in any markup language, but they also include some special syntax so that you can interact with them using Python. The following three constructs mark the specail syntax - {{ }}, {% %} and {# #}. The first construct is used to specify a variable. The second construct specifies an expression and the third one specifies a comment.
